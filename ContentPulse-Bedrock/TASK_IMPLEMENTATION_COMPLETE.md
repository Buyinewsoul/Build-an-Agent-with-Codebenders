## 📋 WHAT USER NEEDS TO DO (25 Minutes)

**Step 1: Create Vercel Account (2 min)**
- Go to https://vercel.com/signup
- Sign up with GitHub
- Authorize access

**Step 2: Deploy Frontend (5 min)**
- Go to https://vercel.com/new
- Import `Build-an-Agent-with-Codebenders`
- Set root directory to `ContentPulse-Bedrock/frontend`
- Click Deploy

**Step 3: Create Render Account (2 min)**
- Go to https://render.com/register
- Sign up with GitHub
- Authorize access

**Step 4: Deploy Backend (10 min)**
- Go to https://dashboard.render.com
- Create Web Service
- Select repo
- Add AWS credentials
- Click Create

**Step 5: Connect Them (3 min)**
- Update Vercel env variable with Render URL
- Redeploy frontend

**Step 6: Test (3 min)**
- Visit Vercel URL
- Test chat
- Verify AI responds
