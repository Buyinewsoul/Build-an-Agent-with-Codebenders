### CHOICE 2: SSH Key (Most Secure)
**Time: 5 minutes**

```bash
# 1. Generate key
ssh-keygen -t ed25519 -C "your_email@example.com"
# Press Enter 3 times

# 2. Start SSH agent
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519

# 3. Add to GitHub (MANUAL)
# Open: https://github.com/settings/ssh/new
# Paste: cat ~/.ssh/id_ed25519.pub
# Click: Add SSH key

# 4. Configure git remote
cd "ContentPulse-Bedrock"
git remote set-url origin git@github.com:Buyinewsoul/Build-an-Agent-with-Codebenders.git

# 5. Push
git push -u origin main

# 6. Verify
open https://github.com/Buyinewsoul/Build-an-Agent-with-Codebenders
```
