## 🔒 ERROR HANDLING

If you see errors:

**"Backend connection failed"**
- Check Terminal 1 - is backend still running?
- If not: `npm start` in backend directory

**"Module not found"**
- Check browser console (F12)
- All imports should be fixed
- If issues persist, restart React (Ctrl+C, npm start)

**"Claude didn't respond"**
- Normal - give it 20+ seconds
- Check backend logs for AWS errors
- Could be rate limiting on AWS
