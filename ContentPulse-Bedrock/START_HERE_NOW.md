## 🚨 IF SOMETHING GOES WRONG

### Backend won't start?
- Check Node.js installed: `node --version`
- Check port 5001 free: `lsof -i :5001`
- Check .env exists: `ls backend/.env`
- Reference: EXECUTE_NOW.md

### Curl times out?
- Verify backend is running
- Check output from Terminal 1
- Wait 5-10 seconds, try again
- Reference: RESPONSE_PARSER.md

### "Unexpected field type" error?
- Check config.js lines 17-21
- Verify fix is there
- Contact immediately
- Reference: FIX_DOCUMENTATION.md

### Response invalid JSON?
- Check curl command spelling
- Copy response exactly as shown
- Try again
